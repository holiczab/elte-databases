# DB1 Practice Guide

### Part 1: Basic Topics (SQL)

**Practice 1: SQL Fundamentals**
*   **Goal:** Introduce the relational model and basic data retrieval.
*   **Topics Covered:**
   *   **Relational Model**
   *   **Basic SELECT Statement**

**Practice 2: Single-Row Functions**
*   **Goal:** Master data manipulation and formatting for individual rows.
*   **Topics Covered:**
   *   **Character Functions**
   *   **Number Functions**
   *   **Date Functions**
   *   **Conversion & General Functions**

**Practice 3: Aggregating and Grouping Data**
*   **Goal:** Learn to summarize data and analyze it in groups.
*   **Topics Covered:**
   *   **Aggregate Functions**

**Practice 4: Combining Multiple Datasets**
*   **Goal:** Understand how to combine rows and tables.
*   **Topics Covered:**
   *   **Joins**
   *   **Set Operators**

**Practice 5: Advanced Queries and Views**
*   **Goal:** Build complex queries and create reusable virtual tables.
*   **Topics Covered:**
   *   **Subqueries**
   *   **Views**
   *   **Relational Algebra**

**Practice 6: Data Definition, Manipulation, and Control**
*   **Goal:** Learn to create database objects, modify data, and manage transactions.
*   **Topics Covered:**
   *   **DDL (Data Definition Language)**
   *   **DML (Data Manipulation Language)**
   *   **Transactions**
